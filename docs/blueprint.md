# **App Name**: ScholarlyFlow

## Core Features:

- Secure User Authentication & Role Management: Enables user login and registration, managing distinct access permissions for Admin, Teacher, Student, and Parent roles.
- Personalized Dashboards: Displays role-specific overviews with key information relevant to Admin (totals, statistics), Teacher (schedules, classes), Student (grades, upcoming exams), and Parent (child's progress, fee status).
- Student & Teacher Profile Management: Allows administrators to create, view, and update student and teacher accounts and their basic information.
- Class & Subject Configuration: Enables administrators to create and manage academic classes, assign subjects, and enroll students and teachers.
- Assignment Creation & Submission System: Teachers can create and assign tasks, and students can view and submit their completed assignments online.
- Grade Management & Reporting: Teachers can input and manage grades for assignments and track overall student performance.
- AI-Enhanced Student Progress Overview: A generative AI tool analyzes student grades and attendance data to provide concise textual summaries of their academic progress and highlight areas for improvement, visible on teacher and parent dashboards.

## Style Guidelines:

- Primary color: A vibrant, professional indigo-purple (#6126D9). This will form one end of a soft gradient for main UI elements.
- Background color: A very light, subtle grayish-purple (#F0F0F7) to ensure cleanliness and spaciousness on dashboards.
- Accent color: A bright, clear blue (#3BB0EB) to draw attention to interactive elements, status indicators, and call-to-action buttons.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for data-rich dashboards and various text lengths.
- Use a consistent set of clean, modern outline icons for sidebar navigation, status badges, and interactive elements, enhancing clarity and user experience.
- Spacious and responsive design, adapting seamlessly to desktop, tablet, and mobile screens. Key elements include sidebar navigation, a top navbar, smooth shadows for depth, and rounded card components with subtle glassmorphism effects for a contemporary feel.
- Implement subtle animations for transitions between pages and interactions (e.g., hover effects on buttons, smooth state changes) to provide a fluid and elegant user experience without being distracting.
